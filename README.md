## üß† Key Insights from Analysis
* **Overtime Impact**: Employees working frequent overtime showed a significantly higher likelihood of leaving.
* **Career Stagnation**: A strong correlation was observed between lack of promotions (in the last 3-4 years) and attrition.
* **Income Sensitivity**: Lower salary slabs exhibited higher churn rates, suggesting a need for compensation review.
